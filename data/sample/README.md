# Sample Demo Data

This folder contains a small hand-built demo dataset so the Streamlit app can show the recommender experience before large external datasets are downloaded.

The sample data is not scraped from Google Maps, Yelp pages, TripAdvisor, or any review-site page. It is synthetic portfolio demo text written to exercise the recommender dimensions. Real project runs should rebuild `data/processed/` from permitted sources such as the Yelp Open Dataset, OpenStreetMap, City of Vancouver Open Data, and optional Reddit API text.
